BIENVENIDO



Este OVI tiene como propósito identificar y aprender las diferentes etapas del software, Desarrollar un software significa construirlo simplemente mediante su descripción, para ello debemos utilizar una serie de pasos y métodos ya establecidos, para realizar un correcto desarrollo y así evitar que este se vuelva mucho más complejo ya que si establecemos muy bien cada etapa tendremos una base firme que no cambiaremos durante el trascurso de su desarrollo.
